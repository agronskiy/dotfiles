# CLAUDE.md Management

Never edit `~/.claude/CLAUDE.md` directly — it is auto-generated by `claude-setup`.

Always edit the source files, then run `claude-setup` to regenerate.

## Before editing: confirm visibility

Always ask: "Should this go in **public** or **local** CLAUDE.md?"

- **Public** (fine in a public repo): edit `~/.dotfiles/.claude/CLAUDE.md`
- **Local** (internal tools, private context): edit `~/.dotfiles-local/.claude/CLAUDE.md`

# Always regenerate merged output
claude-setup

claude-setup --status
